# npm star

**标记喜欢的npm包**

### 梗概

```shell
npm star [<pkg>...]
npm unstar [<pkg>...]
```

### 描述

给一个npm包 **star** 表示你对这个npm包有兴趣。这是一种稍微积极的方式表示你对这个包的关注。

撤销对一个npm包的 **star** 是一样的，只不过意思是相反的。

这是一件只有两面(喜欢或者不喜欢)的事情。重复给一个npm包 **star** 没有额外的效果，即 **star** 只能给一次。
