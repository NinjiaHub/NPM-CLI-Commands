# npm stars

**查看所有标记为喜欢(star)的包**

### 梗概

```shell
npm stars [<user>]
```

### 描述

用来查看以前给过 **star** 的所有的npm包。

如果想知道朋友给过哪些npm包 **star**，也可以用这条命令来实现。
