# npm ping

**ping npm 注册表**

### 梗概

```shell
npm ping [--registry <registry>]
```

### 描述

该命令会 **ping** 配置好的、或者命令行中指定的 **npm 注册表** 然后验证身份。如果身份验证通过则会输出类似下面的信息：

```shell
> Ping success: {*Details about registry*}
```

否则：

```shell
> Ping error: {*Detail about error}
```
