# npm prefix

**显示路径前缀**

### 梗概

```shell
npm prefix [-g]
```

### 描述

将 **局部前缀** 打印到标准输出。如果没有使用 **-g** 选项，**prefix** 的值为包含 **package.json** 文件的目录的路径。

如果使用了 **-g** 选项，将会打印 **全局范围的前缀**。详情请戳👉[npm-config](https://docs.npmjs.com/misc/config "npm-config")
